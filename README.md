# NullSec Toolkit

Multi-tool web application for security, CTF, Linux reference, and GitHub stats.

## Features

### Security Toolkit
- Hash Generator (MD5, SHA256, SHA512)
- Base64 Encode/Decode
- Password Generator
- Port Scanner (simulated)
- IP Info Lookup
- QR Code Generator

### Linux Command Cheat Sheet
- 70+ Linux commands searchable by name, description, or category
- Categories: file, text, process, disk, network, archive, user, package, system, help
- Each command includes example usage

### GitHub Stats Dashboard
- View user info, repo stats, total stars/forks
- Top languages breakdown
- Recent pull requests list

### CTF Toolkit
- Cipher decoder (ROT13, ROT47, Atbash, Hex, Binary, Morse)
- Number base converter (dec/hex/oct/bin)
- XOR brute force decryption
- Hash type identifier
